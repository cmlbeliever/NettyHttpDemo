# 基于Netty的仿SpringMvc服务器 #
## 实现功能 ##
1. 支持静态文件访问
2. 支持表单提交处理
3. 支持重定向

## 程序入口 ##
- NettyHttpServer：为程序入口，运行后即可启动服务
- 访问地址为localhost:8080
